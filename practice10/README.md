# FreshFood — лендинг сервиса доставки здорового питания

## Тема проекта
Сервис доставки здорового питания с готовыми рационами на день.

## Используемый CSS-фреймворк
**Bulma** (версия 1.0.0)

## Структура проекта
- `index.html` — Главная страница
- `contacts.html` — Страница контактов и отзывов (содержит форму обратной связи)
- `faq.html` — Страница часто задаваемых вопросов
- `404.html` — Страница 404 (не найдено)
- `style.css` — Основные стили
- `scripts/` — Папка с JavaScript-файлами
  - `validation.js` — Скрипт валидации формы
  - `consoleLogger.js` — Скрипт вывода данных формы в консоль
- `README.md` — Документация

## Практическое занятие 10: Валидация форм и работа с событиями
